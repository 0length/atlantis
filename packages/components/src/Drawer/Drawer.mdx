# Drawer

<ComponentStatus stage="pre" responsive="no" accessible="no" />

Drawer is a UI element meant to divide to areas of content.

```ts
import { Drawer } from "@jobber/components/Drawer";
```

## Design & Usage Guidelines

The `Drawer` component comes with no spacing around it. This means that it
should be used within a [`Content`](/components/content) component.
