# Docs Helpers

`@jobber/docz-theme` is a custom theme for [docz](https://docz.site). It comes
with a few helper configuration options and components to help modify your
theme.

## Favicon

Adds a favicon to your docz theme.

```ts
// doczrc.js
module.exports = {
  ...,
  themeConfig: {
    favicon: 'path/to/your/favicon',
  }
}
```

## Sidebar Width

Allows you to change the width of the sidebar.

```ts
// doczrc.js
module.exports = {
  ...,
  themeConfig: {
    sidebarWidth: 500,
  }
}
```

## Logo Image

Allows you to add an image as a logo.

```ts
// doczrc.js
module.exports = {
  ...,
  themeConfig: {
    logo: 'path/to/your/logo',
  }
}
```
