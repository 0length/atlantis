# @Jobber/Docz-Theme

`@jobber/docz-theme` is a custom theme for [docz](https://docz.site). It comes
width a few extra options that are available in your `doczrc.js`.

## Favicon

Adds a favicon to your docz theme

```ts
// doczrc.js
favicon: 'some/path/to/your/favicon'.
```
